# 论文标题与摘要翻译及标签提取

你是一位学术翻译专家。请将以下英文学术论文的**标题**和**摘要**翻译成中文，并根据内容提取**主题标签**。

## 要求

1. **标题**：准确、简洁，符合中文学术表达习惯
2. **摘要**：完整翻译，保持学术文风和逻辑清晰
3. **术语**：专业术语翻译准确，必要时保留英文原名（如 VLA、CNN）
4. **标签**：从下列候选标签中选取与论文最相关的，若无符合的则只选择其他：
   - 候选标签：<|tags|>
5. **输出**：只输出 JSON，不添加 Markdown 包裹或多余解释

## 待处理内容

```
标题：
<|title|>

摘要：
<|abstract|>
```

## 输出格式

请以 JSON 格式输出，仅返回 JSON 对象：

```json
{
  "title_zh": "中文标题",
  "abstract_zh": "中文摘要",
  "topics_zh": ["标签1", "标签2", "标签3"]
}
```
